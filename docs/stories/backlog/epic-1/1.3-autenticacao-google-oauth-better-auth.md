## User Story

Como usuário,
quero entrar com minha conta Google,
para que eu acesse a plataforma sem criar senha.

## Acceptance Criteria

- [x] Botão de login com Google disponível
- [x] better-auth configurado com provedor Google OAuth
- [x] Sessão persistida com cookie ou token
- [x] Logout funcional
- [x] Redirecionamento para o Studio após login

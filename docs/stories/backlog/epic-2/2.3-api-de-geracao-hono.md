# Story 2.3: API de Geração (Hono)

## User Story

Como desenvolvedor,
quero uma rota de geração em Hono,
para que o frontend tenha um endpoint confiável para criar imagens e salvar metadados.

## Acceptance Criteria

- [ ] Rota POST `/api/generate-image` recebe `style` e `prompt`
- [ ] Prompt otimizado para line-art construído server-side
- [ ] Chamada ao AI Gateway com resposta de imagem ou equivalente suportado
- [ ] Upload do resultado para o storage
- [ ] Metadados registrados em `userImages`
- [ ] Rate limiting por usuário aplicado

## Scope

- In scope: endpoint, orquestração, upload e persistência
- Out of scope: UI de geração e galeria

## Tasks

- [ ] Implementar o endpoint e a construção do prompt
- [ ] Integrar AI Gateway, storage e banco

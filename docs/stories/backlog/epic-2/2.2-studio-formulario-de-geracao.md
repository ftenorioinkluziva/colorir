## User Story

Como usuário,
quero escolher um estilo e escrever um prompt,
para que eu gere uma line-art personalizada.

## Acceptance Criteria

- [ ] Seletor visual com 4 estilos
- [ ] Campo de texto para prompt livre
- [ ] Botão Gerar com estado de loading
- [ ] Validação de limite diário de gerações
- [ ] Chamada para API Hono que orquestra o AI Gateway
- [ ] Tratamento de timeout, falha do provedor e conteúdo bloqueado

## Scope

- Out of scope: galeria, seleção em lote e exportação PDF

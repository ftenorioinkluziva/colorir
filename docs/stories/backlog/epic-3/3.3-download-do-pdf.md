# Story 3.3: Download do PDF

**ID:** 3.3 | **Epic:** Epic 3: Motor de Exportação PDF
**Order:** 12/12 | **Sprint:** 3 | **Points:** 3 | **Priority:** 🟡 Medium
**Status:** Ready for Review
**Source:** [docs/prd.md](../../../prd.md)

## User Story

Como usuário,
quero baixar o PDF gerado com rapidez e segurança,
para que eu possa imprimir e colorir offline.

## Acceptance Criteria

- [ ] Download direto via URL assinada do MinIO
- [ ] Nome do arquivo segue o padrão `colorir-{timestamp}.pdf`
- [ ] Feedback de progresso durante a geração
- [ ] Erro mostrado quando o PDF exceder o limite de tamanho

## Scope

- In scope: entrega do arquivo, naming e feedback
- Out of scope: geração e preview

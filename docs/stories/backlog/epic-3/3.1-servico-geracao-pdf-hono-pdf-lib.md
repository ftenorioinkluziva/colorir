# Story 3.1: Serviço de Geração de PDF (Hono + pdf-lib)

## User Story

Como desenvolvedor,
quero um serviço server-side de geração de PDF,
para que as imagens selecionadas sejam compiladas em um arquivo pronto para impressão.

## Acceptance Criteria

- [ ] Rota POST `/api/generate-pdf` recebe um array de `imageIds`
- [ ] Documento A4 é criado com `pdf-lib`
- [ ] Cada imagem ocupa uma página própria
- [ ] Quebra de página automática entre imagens
- [ ] PDF salvo no storage
- [ ] Registro persistido em `userPdfs`
- [ ] Erros de imagem ausente ou falha do `pdf-lib` tratados

## Scope

- In scope: geração server-side, persistência e erro
- Out of scope: preview e download

## Tasks

# Story 3.2: Preview do PDF

**ID:** 3.2 | **Epic:** Epic 3: Motor de Exportação PDF
**Order:** 11/12 | **Sprint:** 3 | **Points:** 5 | **Priority:** 🟡 Medium
**Status:** 📋 Draft
**Source:** [docs/prd.md](../../../prd.md)

## User Story

Como usuário,
quero ver uma prévia do PDF antes de baixar,
para que eu confirme a seleção e a ordem das páginas.

## Acceptance Criteria

- [ ] Preview com miniaturas das páginas na ordem
- [ ] Reordenação das imagens antes de gerar
- [ ] Botão para confirmar e baixar PDF
- [ ] Estado vazio quando não houver imagens

## Scope

- In scope: preview, reordenação e confirmação
- Out of scope: geração server-side e entrega final

## Tasks

- [ ] Renderizar miniaturas e ordem das páginas
- [ ] Permitir reordenação antes da exportação
- [ ] Conectar o fluxo de confirmação ao download
